ОС разработки Windows 7 Service Pack 1


Версия интерпретатора Python 3.5.2

Библиотеки: только Flask

Структура решения:

kua_adcombo
|
----задача о сервере, вычисляющем перестановки
    |
    ----static  статические файлы
    |
    ----t3routine   вспомогательные модули
    |
    ----templates   шаблоны
    |
    ----server.py   модуль запуска сервера


Замечания:

1) в модуле запуска сервера 
server.py объявлена 'псевдо'-константа FLASK_RUN_IN_DEBUG_MODE (в поставке  
равна True), которая отвечает за режим debug при запуске сервера Flask.

2) сервер настроен на локальный хост с портом 5000 по умолчанию.
